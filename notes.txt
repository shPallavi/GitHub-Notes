🧠 What is Version Control System (VCS)?

👉 A Version Control System (VCS) is a tool that helps you:

Track changes in your code
Go back to previous versions
Work with multiple developers

💡 Simple Example

Imagine you are working on a file:

project.js

Without VCS:

You overwrite code ❌
You lose old versions ❌

With VCS:

Every change is saved ✅
You can go back anytime ✅


What VCS Does

✔ Keeps history of code
✔ Shows who changed what
✔ Allows collaboration
✔ Prevents data loss

🛠 Popular VCS

👉 The most popular one is:

Git

🐙 What is Git?

👉 Git is a Version Control System (tool installed on your computer)

✔ It helps you:
Track code changes locally
Create branches
Merge code
Manage versions

🌐 What is GitHub?

👉 GitHub is a cloud platform for Git repositories

| Feature | Git             | GitHub                       |
| ------- | --------------- | ---------------------------- |
| Type    | Tool            | Platform                     |
| Works   | Local system    | Cloud                        |
| Purpose | Version control | Code hosting & collaboration |

🧠 Easy Way to Understand

👉 Git = engine
👉 GitHub = cloud storage + collaboration

Real Workflow
Write Code → Git → Push → GitHub

git init        # start project
git add .       # add files
git commit -m "message"
git push        # upload to GitHub


🚀 Most Important Git Commands

🔰 1. Setup (One-time)

git config --global user.name "Your Name"
git config --global user.email "your@email.com"

👉 Sets your identity

📁 2. Initialize Repository

git init

👉 Starts Git in your project

📥 3. Clone Repository

git clone https://github.com/user/repo.git
👉 Download project from GitHub

➕ 4. Add Files

git add .

👉 Adds all files to staging

git add file.js

👉 Add specific file

💾 5. Commit Changes

git commit -m "your message"

👉 Save changes with message

🚀 6. Push Code (Upload)

git push origin main

👉 Send code to GitHub

📥 7. Pull Code (Download Latest)

git pull origin main

👉 Get latest updates

🔍 8. Check Status

git status

👉 See:

changed files
staged files

📜 9. View Commit History

git log

👉 Shows all commits


🌿 10. Branching (Very Important)

Create branch

git branch feature

Switch branch

git checkout feature

👉 Or modern way:

git switch feature

11. Merge Branch

git merge feature

👉 Merge feature into current branch

❌ 12. Undo Changes

Unstage file

git reset file.js

Undo last commit

git reset --soft HEAD~1

🧹 13. Remove File

git rm file.js
